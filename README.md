This folder is convertion of most usefull bash command for windows.

INSTALLATION:
Add folder location to PATH.
For telnet and ssh install putty.

BUILD IN COMMANDS:

cat -> dir
clear -> cls
cp -> copy
diff -> fc
grep -> find
ls -> dir
mv -> rn
rm -> del

EXTRA COMMANDS:

ssh -> putty -ssh
telnet -> putty -telnet

ADDONS:
rand -> Generates random 16 length string
loop -> Loops command for n times
